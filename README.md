<h3 align="center">
  :checkered_flag: Startando o projeto
</h3>

1° Clonar: `git clone https://github.com/leopacciulli/Bootcamp-Desafio-NodeJS.git`

<br>

2° Rodar o back end em http://localhost:3333:

```sh
$ cd Bootcamp-Desafio-NodeJS 
$ npm install
$ yarn/npm start
```

3° Rodar os testes:

```sh
$ cd Bootcamp-Desafio-NodeJS 
$ yarn/npm test
```
